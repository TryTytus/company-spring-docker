# Installation and Startup

### clone repository

```bash
git clone https://github.com/TryTytus/company-spring-docker
```
### cd to dir

```bash
cd company-spring-docker
```

### run with docker compose

> ![IMPORTANT]
>  startup may take some time


```bash
docker compose up
```
